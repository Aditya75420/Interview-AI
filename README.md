# 🚀 PrepWise AI  

Smarter interview practice with AI-powered feedback.

---

## 🌐 Live Demo  

- **Frontend:** https://interview-ai-ten-liart.vercel.app  
- **Backend:** https://interview-ai-s7nu.onrender.com  

---

## ✨ Features  

- 🔐 User Authentication (Signup/Login)  
- 🤖 AI-powered interview assistance  
- ⚡ Real-time response evaluation  
- 🌍 Fully deployed full-stack application  

---

## 🛠️ Tech Stack  

- **Frontend:** React (Vite)  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Deployment:** Vercel, Render  

---

## ⚙️ Environment Variables  

### Backend  
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
GOOGLE_GENAI_API_KEY=your_api_key
CLIENT_ORIGIN=https://your-frontend-url.vercel.app

### Frontend  
VITE_API_URL=https://your-backend-url.onrender.com

```bash
git clone https://github.com/Aditya75420/INTERVIEW-AI.git
cd INTERVIEW-AI

# Backend

# Frontend
